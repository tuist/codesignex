# Codesignex

Codesignex is an Elixir library to sign bundles generated for Apple platforms.

## Installation

If [available in Hex](https://hex.pm/docs/publish), the package can be installed

```elixir
def deps do

    {:codesignex, "~> 0.1.0"}
  ]
end
```
